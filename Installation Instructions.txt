SimuSim: Simultaneous Power Analysis for a Set of Statistical Tests Installation Instructions

To install this package, copy-and-paste the text, below, into R.

# Install the devtools package, if you don't already have it installed, which allows you to install packages from GitHub.
install.packages("devtools")

# Load the devtools library
library(devtools)

# Install SimuSim
install_github("joelleforestier/SimuSim")

# Load SimuSim
library(SimuSim)
