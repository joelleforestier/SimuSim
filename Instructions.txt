SimuSim: Simultaneous Power Analysis for a Set of Statistical Tests Instructions

TO INSTALL THIS PACKAGE:

# Install the devtools package (if you don't already have it installed) which allows you to install packages from GitHub
install.packages("devtools")

# Install SimuSim
devtools::install_github("joelleforestier/SimuSim")

TO USE THIS PACKAGE:

# Load SimuSim
library(SimuSim)

# If you want to assess power for multiple tests contained in a single model (e.g., separate predictors in a multiple regression), check the help page for simusim.multivar:
?simusim.multivar

# If you want to assess power for multiple tests contained in separate models (e.g., multiple t-tests), check the help page for simusim.bivar:
?simusim.bivar

TO CITE THIS PACKAGE:

When using this function, please cite as: Le Forestier, J. M., Page-Gould, E., & Chasteen, A. L. (Forthcoming). Statistical power for a set of tests.
